# labmailu
Pour installer un serveur MAILU simplement, suivez les instructions suivantes :
<P></P>
sudo su
<P></P>
mkdir /mailu
<P></P>
cd /mailu
<P></P>
git clone https://github.com/projetpigier/labmailu.git
<P></P>
cd labmailu
<P></P>
docker compose up -d
<P></P>
